Language Learning & Education

Education is at the heart of what we do. Our flagship products in this space are designed to make learning accessible, engaging, and effective.

Vietnamy — Vietnamese Language Learning Platform

Vietnamy is our answer to a simple question: why is there no dedicated app for learning Vietnamese? Built with React 19 and a custom Express.js backend, Vietnamy delivers:

  • Interactive lessons with gamification (coins, hearts, streaks)
  • Spaced repetition (SM-2 algorithm) for vocabulary retention
  • Comprehensive dictionary with IPA transcription
  • Grammar drills and pronunciation training for all 6 Vietnamese tones
  • Bilingual UI supporting English and Chinese learners
  • Full admin CMS for content management

The architecture is designed for adaptation — the same codebase can be extended to other language pairs with minimal modification.

Learny — Full-Stack Educational Monorepo

Learny demonstrates our approach to scalable education platforms. Built as a monorepo with Next.js frontend and NestJS backend, it features Prisma ORM for type-safe database access and a clean separation of concerns between web application and API.

Edu-ERP — Educational Resource Planning

For institutions managing complex educational operations, Edu-ERP provides calendar integration, form validation, analytics dashboards with Recharts, and smooth animations via Framer Motion — all built on Next.js 16 with React 19.

Interactive 3D & Creative Technology

We push the boundaries of what's possible in the browser. Our 3D projects showcase advanced graphics programming and real-time audio-visual experiences.

Sound Voyage — Audio-Reactive Space Shooter

Sound Voyage (sdn-viz) is our most technically ambitious project: a fully audio-reactive 3D space shooter game. Features include:

  • Real-time FFT audio analysis and beat detection via Web Audio API
  • Custom GLSL vertex and fragment shaders for particle systems
  • Three.js rendering with React Three Fiber integration
  • Post-processing effects: Bloom, Vignette, and chromatic aberration
  • Object pooling for 60fps performance with thousands of particles
  • Singleton architecture for AudioEngine and GameState management

The neon spaceship responds to music in real-time — enemies spawn on beats, visual effects pulse with bass, and the starfield reacts to audio intensity.

Statics Lab — Interactive Physics Visualization

Statics Lab brings engineering concepts to life. Using Matter.js for physics simulation, Konva for 2D canvas rendering, and KaTeX for mathematical notation, it provides an interactive environment for learning statics and mechanics.

Our Development Approach

Across all 29 projects, several patterns emerge that define how we build software:

  • Offline-First Architecture — Critical applications work without internet, syncing when connectivity returns
  • Mobile-First Design — Responsive interfaces optimized for touch, designed for one-handed operation
  • Monorepo Structure — Shared code, consistent tooling, and unified dependency management
  • TypeScript Everywhere — Type safety from database to UI for fewer bugs and better developer experience
  • Component Libraries — Radix UI for accessibility, shadcn/ui for rapid development
  • Gamification — Points, streaks, and rewards where they enhance engagement
