What if your music could become a game? Sound Voyage transforms any audio into an immersive space shooter experience. Every beat spawns enemies, every bass drop triggers visual explosions, and every melody shapes the starfield around you. Built entirely in the browser using cutting-edge WebGL technology.

The Experience

Sound Voyage is a synesthetic journey where audio and visuals merge:

  • Play any music — Upload your own tracks or use built-in audio
  • Pilot your ship — Navigate through a procedurally generated starfield
  • Fight the beat — Enemies spawn and attack in sync with the music
  • Feel the bass — Visual effects pulse and bloom with audio intensity
  • Chase high scores — Your performance is scored based on survival and accuracy

Technical Innovation

Sound Voyage showcases what's possible in modern browsers:

Real-Time Audio Analysis

  • Web Audio API — Direct access to audio streams for analysis
  • FFT Processing — Real-time frequency decomposition
  • Beat Detection — Custom algorithms identify rhythmic events
  • Frequency Band Mapping — Different frequency ranges drive different game elements

Advanced 3D Graphics

  • Three.js + React Three Fiber — Declarative 3D scene composition
  • Custom GLSL Shaders — Hand-written vertex and fragment shaders for particle systems
  • Post-Processing — Bloom, vignette, chromatic aberration for that neon aesthetic
  • 60fps Performance — Object pooling and GPU optimization for smooth gameplay

Architecture

  • Singleton Pattern — AudioEngine and GameState as single sources of truth
  • React 19 + Vite — Modern build pipeline with hot module replacement
  • TypeScript — Type safety for complex game logic
  • Entity Component System — Efficient game object management

How Audio Drives Gameplay

Enemy Spawning

Beat detection triggers enemy waves. Fast-paced music means more enemies. The game literally plays to your music.

Visual Effects

Low frequencies drive bloom intensity. High frequencies sparkle the starfield. Mid-range shapes particle trails.

Ship Response

Your ship's glow, trail length, and shield effects respond to audio energy. You become part of the visualization.

Difficulty Scaling

Intense music sections increase difficulty. Quiet passages offer brief respite. The game pacing matches the music pacing.
