Engineering statics is the foundation of structural analysis, mechanical design, and civil engineering. Yet for generations, students have learned it from static textbook diagrams that can't show how forces actually interact. Statics Lab changes that — an interactive learning environment where you can build, test, and understand mechanical systems in real-time.

The Learning Challenge

Statics is notoriously difficult to learn from books:

  • Abstract concepts — Forces, moments, and equilibrium are invisible
  • 2D limitations — Textbook diagrams can't show dynamic behavior
  • Trial and error missing — You can't experiment with a printed page
  • Math disconnect — Equations don't intuitively connect to physical reality

Students memorize formulas without understanding. They pass exams but struggle with real engineering problems.

Statics Lab: Learning by Doing

Statics Lab provides an interactive sandbox for exploring mechanics:

Build Systems

  • Drag-and-drop construction — Create beams, supports, and connections
  • Apply forces — Add point loads, distributed loads, moments
  • Define constraints — Pins, rollers, fixed supports
  • Connect members — Build trusses, frames, and machines

See Forces

  • Real-time visualization — Watch force vectors change as you modify the system
  • Reaction forces — See how supports respond to loading
  • Internal forces — Visualize shear, moment, and axial force diagrams
  • Equilibrium indicators — Know instantly if your system is balanced

Understand Math

  • Live equations — KaTeX rendering shows the math behind the physics
  • Step-by-step solutions — See how equilibrium equations are derived
  • Variable exploration — Change values and watch equations update
  • Unit handling — Proper engineering units throughout
